How to create a animation DLL object

Implementing CreateCAnimDllL()

Use CreateCAnimDllL() to construct the CAnimDll derived server side DLL class.

For example, for a DLL class CExampleAnimDll the following code must appear in the DLL source file:

// the only exported function in the DLLIMPORT_C CAnimDll* CreateCAnimDllL();

EXPORT_C CAnimDll* CreateCAnimDllL(){return new(ELeave) CExampleAnimDll;}

The use of both IMPORT_C and EXPORT_C statements is required to satisfy different compiler conventions.
